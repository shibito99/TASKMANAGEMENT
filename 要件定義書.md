# 要件定義書：タスク管理アプリ（Trelloライク）

**バージョン：** 1.0  
**作成日：** 2026-04-24  
**ステータス：** ドラフト

---

## 1. プロジェクト概要

- **プロジェクト名：** タスク管理アプリ
- **目的：** ボード・リスト・カードでタスクを視覚的に管理する
- **対象ユーザー：** 個人利用（自分自身）
- **動作環境：** ローカル環境（ブラウザで起動）
- **使用技術：** React + Vite + TypeScript + Tailwind CSS

---

## 2. 用語定義

- **ボード：** アプリ全体の作業スペース（Trelloのボードに相当）
- **リスト：** タスクをグループ化する列（例：To Do / In Progress / Done）
- **カード：** 個々のタスク（Trelloのカードに相当）

---

## 3. 機能要件

### ボード機能

- **B-01** ボードの表示 `Must`
  - リストとカードを一覧表示する
  - 初期リリースはボードは1つのみ（複数ボードは対象外）

---

### リスト機能

- **L-01** リストの追加 `Must`
  - 任意の名前でリストを新規追加できる
- **L-02** リストの削除 `Must`
  - リストを削除できる（内包するカードも同時に削除）
- **L-03** リスト名の編集 `Must`
  - リスト名をクリックして直接編集できる
- **L-04** リストの並び替え `Want`
  - ドラッグ&ドロップでリストの順序を変更できる

---

### カード機能

- **C-01** カードの追加 `Must`
  - リスト内に任意のタイトルでカードを追加できる
- **C-02** カードの削除 `Must`
  - カードを削除できる
- **C-03** カードタイトルの編集 `Must`
  - カードのタイトルを編集できる
- **C-04** カードの移動 `Must`
  - ドラッグ&ドロップでカードを別リストへ移動できる
- **C-05** カードの詳細表示 `Want`
  - カードをクリックすると詳細モーダルを表示できる
- **C-06** 説明文の追加 `Want`
  - カード詳細に説明文を記入できる
- **C-07** 期限日の設定 `Want`
  - カード詳細に期限日を設定できる
- **C-08** ラベルの付与 `Want`
  - カードに色付きラベルを付けられる

---

### データ保存機能

- **D-01** データの永続化 `Must`
  - ページを再読み込みしてもデータが消えない
  - localStorage を使用して保存する

---

## 5. データ構造

```typescript
// ボード
Board {
  id: string
  lists: List[]
}

// リスト
List {
  id: string
  title: string
  cards: Card[]
}

// カード
Card {
  id: string
  title: string
  description: string     // Want
  dueDate: string | null  // Want
  label: string | null    // Want
}
```

---

## 6. 非機能要件

- **動作ブラウザ：** Google Chrome 最新版
- **レスポンシブ：** 対象外（PC画面のみ）
- **パフォーマンス：** 特に制約なし（ローカル動作のため）
- **セキュリティ：** 特に制約なし（個人ローカル利用のため）

---

## 7. スコープ外（今回は作らないもの）

- ユーザー認証（ログイン・パスワード管理）
- 複数ボードの管理
- 複数人での共同編集・リアルタイム同期
- スマートフォン・タブレット対応
- サーバーへのデータ保存・クラウド同期
- 通知・リマインダー機能
- ファイル添付機能

---

## 8. 開発フェーズ（実装順序）

### Phase 1 ─ 基本構造の構築
- ボード画面の表示
- リストの追加・削除・名前編集
- カードの追加・削除・タイトル編集

### Phase 2 ─ ドラッグ&ドロップ
- カードのリスト間移動（C-04）

### Phase 3 ─ データ永続化
- localStorage へのデータ保存（D-01）

### Phase 4 ─ カード詳細
- 詳細モーダルの実装（C-05）
- 説明文・期限日・ラベルの追加（C-06〜C-08）
